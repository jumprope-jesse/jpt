## How to develop

1. Clone the repository git clone git@github.com:mayneyao/eidos.git
1. Run pnpm install to install dependencies
1. Run pnpm build (only needed once)
1. Run pnpm dev
1. You can now access the app in your browser at http://localhost:5173
### Generate sdk types

```plain text

```
