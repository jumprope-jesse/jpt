# Find stories in your data

Datasette Cloud is a hosted service that lets your team upload, share and analyze data through an interactive web app and API. It's the SaaS platform built on the Datasette open source project.

## Why use Datasette Cloud?

### Easy to use

Upload your CSV data or SQLite files and Datasette Cloud will do the rest. No coding required.

### Powerful and flexible

Datasette Cloud provides a full-featured app and API for your data, with features like faceting, filtering, sorting, searching, and more.

### Secure and reliable

Datasette Cloud runs on a secure and scalable cloud platform, with automatic backups and updates. You get full control over who can access your data.
