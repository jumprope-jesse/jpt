# AWS AppFabric’s new generative AI feature helps build cross-app experiences (Preview)

## AI Summary (from Notion)
- Announcement: AWS AppFabric introduced a new generative AI feature aimed at enhancing cross-application experiences.
- Release Status: The feature is currently in Preview mode.
- Purpose: Designed to improve end-user productivity in SaaS applications by generating insights and actions using context from multiple applications.
- Functionality:
- Allows developers to embed AWS AppFabric features directly into their application's UI or AI assistants.
- Offers personalized user experiences by surfacing relevant context to enhance user engagement and loyalty.
- Examples of Features:
- Proactively surfaces actionable insights.
- Generates meeting preparation notes, drafts emails, creates tasks, and schedules meetings.
- Integrations: Currently integrates with popular applications such as Asana, Atlassian Jira, Miro, Slack, Smartsheet, Microsoft 365, and Google Workspace.
- Target Audience: Aimed at application developers looking to enrich app experiences and improve user productivity.
- Geographic Availability: The feature is available in the US East (N. Virginia) AWS Region.

Today, AWS AppFabric announced a new feature, AWS AppFabric for productivity, that reimagines end-user productivity in software-as-a-service (SaaS) applications by generating insights and actions with context from multiple applications. AWS AppFabric is a service that connects multiple SaaS applications to enhance security and increase productivity. Since its launch, IT professionals can use AppFabric to automatically normalize application logs from 19 different SaaS applications for improved SaaS observability across their application stack.

Available in Preview, AppFabric’s newest feature helps application developers enrich app experiences by providing generative AI-driven insights based on context from other applications. Developers embed the AWS AppFabric productivity feature directly into their application’s UI or generative AI assistants, maintaining a consistent experience for their end users while surfacing relevant context from other applications. This increased context allows them to provide a more personalized end-user experiences that increases adoption and loyalty, and users benefit from accessing insights they need without interrupting their flow of work. For example, these APIs proactively surface actionable insights, generate meeting preparation notes, draft emails, create tasks, and schedule meetings based on the context and content from across a user’s applications. Today, AWS AppFabric productivity integrates data from Asana, Atlassian Jira suite, Miro, Slack, Smartsheet, Microsoft 365, and Google Workspace. Applications such as Asana, Miro, and Smartsheet are adopting AppFabric for productivity to help their users prioritize their workday and complete tasks faster.

AWS AppFabric productivity feature is available in Preview today in the US East (N. Virginia) AWS Regions.
