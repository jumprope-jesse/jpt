## Content (from Notion)

Amazon SageMaker Canvas is a no-code tool to build, customize and deploy machine learning (ML) and Generative AI (GenAI) models. Amazon SageMaker Canvas now features a revamped home page that helps customers get started with their ML and GenAI journey faster. You can view and understand the core capabilities the moment you land in SageMaker Canvas, and thus you can reach the most relevant user journey for you, faster.

Starting today, SageMaker Canvas offers an improved homepage that showcases how a user can start with data preparation, model building, GenAI model comparison, and ready-to-use model selection faster. At the same time, a user can get 1-click access to existing data flows and models they have created and help pick up where they left off. The user can also get visibility into the latest product releases and blogs that showcase various use cases and advanced topics to help them achieve more with SageMaker Canvas.

To get started, access the latest version of SageMaker Canvas. A new user can access the latest version by directly launching SageMaker Canvas from their AWS console. An existing user can access the latest version of SageMaker Canvas by clicking “Log Out” and logging back in.

The new homepage experience is available in all AWS regions where Amazon SageMaker Canvas is available today. To learn more, see the SageMaker Canvas service page.
