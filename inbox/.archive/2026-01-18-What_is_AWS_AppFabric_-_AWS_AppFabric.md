# What is AWS AppFabric? - AWS AppFabric

## AI Summary (from Notion)
- Overview of AWS AppFabric: A service that connects SaaS applications within an organization for better management and security.
- Target Users: IT and security teams benefit from a standard schema, while employees can enhance productivity with generative AI.
- Two Main Facets:
- AppFabric for Security: Focused on streamlined management and security of applications.
- AppFabric for Productivity: In preview, utilizes generative AI to improve everyday tasks.
- Key Features:
- No coding required to connect multiple SaaS applications.
- Increases productivity and security across the organization.
- Important Links:
- Pricing details available on the AWS AppFabric Pricing page.
- Supported AWS Regions and endpoints can be found in the AWS General Reference.
