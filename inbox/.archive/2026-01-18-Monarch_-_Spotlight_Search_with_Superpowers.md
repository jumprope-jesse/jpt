### What does Monarch do and what is it for?

Monarch works like a superpowered Spotlight Search, enabling you to do many more things on your computer within a single interface that is simple, yet powerful.

By extending the functionality and user experience of Spotlight, you will be able to accomplish a lot of small, but important tasks throughout your day without breaking your flow, switching apps, or switching workspaces.

### What are the benefits to using Monarch?

There are many benefits to using Monarch, but for brevity, I'll name the first few that come to mind:

1) Everything is easier. While Monarch does quite literally speed up your workflow, what users truly care about is that whatever Monarch is capable of, it is the easiest way to get it done. Monarch is not a command palette app. It's designed to change the UI to best suit the task at hand, and you see this in things like controlling audio devices with the Set Audio Devices command or the Color Picker mode.

3) Cost. Monarch is capable of replacing several different apps for the cost of one app, with a one-time payment. If you find that you only like one aspect of Monarch, there are settings that allow you to use Monarch for only that function.

4) Less things to remember. Monarch is meticulously designed to do two things: reduce cognitive overhead and reduce context switching. Major features (called Modes) can be cycled using the Tab key, and other tools are accessed in the most simple and intuitive ways: To use the calculator start typing "100 + 200", or "5 days from now". ### What exactly is a Superlink?

A Superlink is a powerful way to have instant access to your favorite apps or websites while making it easy to utilize different values or optional values every time.

For example: I have a Superlink set up to search my own projects on Github by typing "git" into Monarch, or another user's profile if I type their username, or navigate to my own profile if I specify neither. If the resulting URL is already open in my browser, Monarch will simply switch to that tab rather than opening a new one, keeping my workspace nice and organized.

Superlinks alone provide enough value to speed up your workflow significantly, and it's very much worth seeing what you can do with them by reading the manual entry here.
