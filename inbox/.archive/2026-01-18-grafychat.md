# grafychat

## AI Summary (from Notion)
- Product Overview: Grafychat is a visual notebook designed for conversational AI, allowing users to manage and organize chats visually.
- Key Features:
- Endless Canvas: Users can create multiple canvases for their conversations and organize them visually.
- Privacy Friendly: Data is stored locally on the user's device, ensuring privacy. No signup is required.
- Control Over Results: Users can apply custom instructions and switch models easily.
- Powerful Search: Full-text search capabilities with precise filtering options.
- Support for Multiple AI Providers: Compatible with OpenAI, Google AI, and more.
- OpenAI Support: Supports all ChatGPT models, including GPT-4, and offers voice input and image generation.
- Customizable Prompt Library: Prompts can be saved, edited, and reused within the app.
- Markdown Editing: Chats function as Markdown editors, allowing users to export conversations easily.
- User Interface: Offers customizable aesthetics, including dark mode and customizable backgrounds.
- Cost: One-time purchase model with three pricing tiers:
- Standard: $39 for all features unlocked.
- Extended: $59 includes self-host package.
- Premium: $79 includes premium support.
- Additional Features: Includes import/export with JSON, keyboard shortcuts, code highlight support, and the ability to pin and color tag chats.
- Future Developments: Desktop application is coming soon, enhancing the functionality of the product.
