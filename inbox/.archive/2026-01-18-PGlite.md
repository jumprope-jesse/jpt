# PGlite

## AI Summary (from Notion)
- PGlite Overview
- A lightweight, embeddable Postgres database designed for local use within applications.
- Created on August 12, 2024.

- Key Features
- Lightweight: Under 3MB when Gzipped, built using WASM.
- Extendable: Supports dynamic extension loading, including pgvector and PostGIS.
- Reactive: Offers built-in support for data loading, synchronization, and live queries.

- Interactive Experience
- Users can create and publish a Postgres database using AI through the platform postgres.new.
- A demo is available to try PGlite directly in the browser.

- Notable Support
- Integrates with extensions like pgvector for enhanced functionality.
- Developed by Supabase, emphasizing community and extension capabilities.
