# Eidos

## AI Summary (from Notion)
- Product Overview: Eidos is an extensible framework for managing personal data in one place throughout a user's lifetime.
- Launch Information: Created on June 22, 2024, the project is not yet started.
- Key Features:
- Privacy and Ownership: Users regain control over their data by merging it from cloud providers into a single local storage.
- Offline Support: Access data without an internet connection, ensuring fast performance.
- Web Application: Runs entirely in the browser with no need for a web server.
- AI Integration: Deep integration with Large Language Models (LLM) for tasks like translation and summarization, with offline capabilities.
- Extensibility: Highly customizable extension system allowing users to build their own unique versions of Eidos.
- Open Source: Code transparency ensures software freedom.
- Open Format: Utilizes open formats for data storage, enabling compatibility with other software.
- Developer Support: Offers an API to extend functionality, making it developer-friendly.
- Email Integration: Unique email addresses for resources allow users to update or create data through email.
- SQLite Standardization: All data is stored in SQLite tables, ensuring data longevity.
- Customization Options: Users can utilize JavaScript and TypeScript for custom functions and data processing logic.
- Community Engagement: Users are encouraged to join the Discord server for updates on development progress.
- Overall Theme: Eidos emphasizes user control, privacy, and flexibility in personal data management.

## Privacy-first.Local-first.

Get your data back from cloud providers and merge them into one place.

You truly own your data, with complete control!

### Offline Support

You can access your data without an internet connection, Data is stored on your local device. It's  blazing fast.

### Web is all you need

Eidos is a web application with no web server.

Everything runs inside your browser.

### AI Features

Deeply integrated with LLM, it makes your life easier with AI. You can translate, summarize, talk to your data, and more - all within Eidos.

AI can also work offline.

### Extensible

Eidos has a highly customizable extension system. You can build your own unique Eidos. It's easy to extend and customize.

### Open Source

Eidos is an open-source project, ensuring that the code is transparent, which is essential for software freedom.

### Open Format

Eidos uses open formats to store data, which means you can access your data with other software.

## Developer Friendly

While Eidos is just a web app with no web server, it also offers an API for developers to extend its functionality. Create your own workflows with ease.

## Email Integration

Capture ideas, notes, and read/watch-it-later easily.

Each resource in Eidos has a unique email address, allowing you to create or update resources simply by sending an email.

## SQLite Standardization

Every table in Eidos is a SQLite table. Access or modify the data using any SQLite client.

SaaS may come and go, but your data in SQLite will always be there.

## Offline AI Capabilities

Download LLM once and use it anytime, even without an internet connection. No data leaves your device.

## Customize Everything

Build your own unique Eidos with an easy and powerful extension system.
