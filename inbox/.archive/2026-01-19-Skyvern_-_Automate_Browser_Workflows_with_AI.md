# Skyvern - Automate Browser Workflows with AI

## AI Summary (from Notion)
- Product Overview: Skyvern is an AI-driven tool designed to automate browser-based workflows.
- Key Features:
- Automation: Works on any website using computer vision and AI.
- Natural Language Processing: Executes complex commands using simple natural language instructions.
- API-Driven: Allows for cloud execution of multiple workflows simultaneously.
- Explainable AI: Provides clear summaries and justifications for AI decisions.
- Proxy Networks: Offers support for targeted proxy usage down to zip-code level.
- CAPTCHA Support: Capable of solving CAPTCHAs to complete workflows.
- Authentication: Supports user account logins, including two-factor authentication.
- Data Extraction: Can extract data in various formats such as CSV or JSON.
- Use Cases:
- Automating procurement pipelines.
- Streamlining government forms.
- Completing workflows in any language.
- Release Date: Created on March 15, 2024.
- Company: Developed by Ikonomos Inc.
- Status: Currently not started.
- Demo Availability: Options to request a demo are provided.
