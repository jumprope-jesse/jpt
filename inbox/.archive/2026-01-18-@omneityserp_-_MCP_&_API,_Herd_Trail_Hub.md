## What is Herd?

Herd is a powerful browser automation platform that lets you connect any AI like ChatGPT, Claude, or even Cursor to your favorite websites. This Trail is a pre-built automation for google.com, bing.com, duckduckgo.com. An MCP (Model Context Protocol) server exposes standard tools to your Agent to read from and interact with google.com, bing.com, duckduckgo.com via your browser securely.

With Herd and this trail, your agent can run:

- search: Search across one or more search engines
