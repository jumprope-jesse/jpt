# Let's Build the GPT Tokenizer [video] | Hacker News

## AI Summary (from Notion)
- The author was previously uninterested in large language models (LLMs) until recently.
- They had an earlier project concept for a no-code app that generated SQL queries and API calls from user interactions with various data sources (MySQL, PostgreSQL, etc.).
- This project was abandoned in 2021 but revisited this year with the idea of leveraging LLMs to infer business insights from data schemas.
- The author has been testing various LLMs (including CodeLlama, Zephyr, Mistral, Llama 2, Claude, and ChatGPT) for their new app.
- Results from tests have been promising, leading the author to believe there is significant potential in using LLMs for generating reports and dashboards directly from data schemas.
- The project aims to automate the process of generating actionable insights through user interactions and data schema analysis.
- The author feels that this approach is no longer niche, as they've identified a relevant problem to solve in the market.
