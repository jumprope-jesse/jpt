## FAQ

### 1. What is Async?

Async is an open-source dev tool that combines AI coding with task management and code review. It's like Claude Code + Linear + GitHub PRs in one opinionated tool.

### 2. What can Async do?

Busywork: bug fixes, UI tweaks, and simple features. Coding agents can't do everything and we think it's a mistake to ask them to do more than they can. Async focuses on the tedious, leaving you free to build what's important.

### 3. Who is Async for?

Engineers. We built Async for ourselves: we're two engineers who've been working in tech for almost a decade now. We care about code, often overly so, and wanted to build a tool that would meet our own standards.

### 4. How is Async different than other coding agents?

We say no. We ask questions in response to ambiguous asks. Async is designed to make the user think; we encourage structured and critical thinking over lazy interfaces that optimize for convenience.

### 5. How does Async work?

Each task in Async is just a GitHub issue. For each task, we run a research agent to confirm requirements and ask clarifying questions. Then we execute tasks in a separate container in the cloud and open a GitHub PR automatically with the changes.

### 6. How much is Async?

We offer a hosted version with a free tier of 10 tasks a day and paid tier of $20 / month.
