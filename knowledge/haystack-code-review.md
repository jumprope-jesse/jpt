# Haystack Code Review Tool

## Overview
Haystack is a code review tool that guides reviewers through pull requests by presenting changes in logical order with rich context.

**Website:** https://haystackeditor.com/

## Key Features

### Logical Change Ordering
Instead of jumping around files, Haystack lays out changes in a logical sequence so reviews stay focused and fast.

### Variable Tracing
Track variables across the entire pull request - from initialization through to cleanup. Helps understand data flow without manually searching.

### Caller Context
For every changed or added function, see all its callers (both new and existing). Useful for understanding impact of changes.

## Value Proposition
- Faster code reviews through better organization
- Reduced context-switching during review
- Better understanding of change impact
- Ship with more confidence
