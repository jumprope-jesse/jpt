# Robot Vacuums

## What It Is
Valetudo is open-source cloud replacement firmware for robot vacuums, enabling **local-only operation** without dependency on manufacturer cloud services.

- **Website**: https://valetudo.cloud
- **Added**: 2026-01-18

## Why It's Interesting
- Privacy: No vacuum data sent to cloud servers
- Local control: Works even if manufacturer shuts down cloud services
- Home Assistant integration potential
- Open source - aligns with ethical tech values

## Buying a Compatible Robot

### Key Questions to Ask
1. What types of floor do you have?
2. Do you need proper mopping?
3. Do you want an Auto-Empty-Dock?
4. Do you need obstacle detection (cables, toys, etc.)?
5. Do you have pets?
6. Do you have people with long hair?
7. What is your budget?
8. Are you comfortable with disassembling a robot?

### Shopping Tips
- Use price comparison websites
- Check shopping communities
- Check Aliexpress for deals
- Support the project via [GitHub Sponsors](https://github.com/sponsors/Hypfer) instead of affiliate links

## Installation
Requires flashing custom firmware - some technical skill needed. Check their docs for supported models.
