# Display Resolution Standards

Quick reference for common display resolutions, aspect ratios, and naming conventions.

## Common Resolutions by Category

### SD / Legacy
| Name | Resolution | Aspect Ratio | Notes |
|------|------------|--------------|-------|
| VGA | 640x480 | 4:3 | Original IBM standard (1987) |
| SVGA | 800x600 | 4:3 | Super VGA |
| XGA | 1024x768 | 4:3 | Extended Graphics Array |

### HD Family
| Name | Resolution | Aspect Ratio | Notes |
|------|------------|--------------|-------|
| HD / 720p | 1280x720 | 16:9 | "HD ready" |
| HD+ | 1600x900 | 16:9 | Also called 900p |
| FHD / 1080p | 1920x1080 | 16:9 | Full HD, ~2 megapixels |
| QHD / 1440p | 2560x1440 | 16:9 | Quad HD (4x 720p pixels) |
| 4K UHD | 3840x2160 | 16:9 | 4x FHD pixels |
| 8K UHD | 7680x4320 | 16:9 | 4x 4K pixels |

### Cinema Standards
| Name | Resolution | Aspect Ratio | Notes |
|------|------------|--------------|-------|
| 2K DCI | 2048x1080 | 256:135 (~1.9:1) | Digital cinema standard |
| 4K DCI | 4096x2160 | 256:135 (~1.9:1) | Cinema 4K |

### Ultrawide (21:9 marketing)

|------|------------|--------------|-------|

| 5K2K | 5120x2160 | 64:27 (2.37:1) | Ultrawide 4K equivalent |

### 16:10 (Common in Laptops/Productivity)
| Name | Resolution | Notes |
|------|------------|-------|
| WXGA | 1280x800 | Common laptop resolution |
| WXGA+ | 1440x900 | 19" widescreen monitors |
| WSXGA+ | 1680x1050 | 22" monitors, 15-17" laptops |
| WUXGA | 1920x1200 | FHD+, good for two-page text |
| WQXGA | 2560x1600 | QHD+, 30" monitors, MacBook 13" Retina |
| WQUXGA | 3840x2400 | UHD+, 4x WUXGA |

## Naming Convention Patterns

- **Q prefix**: Usually "Quad" (4x pixels) - QHD, QXGA, QUXGA
- **W prefix**: "Wide" (16:9 or 16:10 vs 4:3) - WXGA, WUXGA
- **+ suffix**: Slightly higher resolution variant - HD+, SXGA+
- **U prefix**: "Ultra" - UXGA, UHD
- **S prefix**: "Super" - SVGA, SXGA

**Exception**: qHD (lowercase q) = quarter HD (960x540), not quad

## Quick Pixel Math

- 720p to 1080p: 2.25x more pixels (1.5x each dimension)
- 1080p to 4K: 4x more pixels (2x each dimension)
- 4K to 8K: 4x more pixels (2x each dimension)

## Aspect Ratio Reference

| Ratio | Decimal | Common Use |
|-------|---------|------------|
| 4:3 | 1.33 | Legacy CRT, iPad |
| 3:2 | 1.50 | Surface devices, some laptops |
| 16:10 | 1.60 | Productivity monitors, MacBooks |
| 16:9 | 1.78 | HDTV, most monitors |
| 21:9 | 2.33-2.40 | Ultrawide (actual ratios vary) |
| 32:9 | 3.56 | Super ultrawide |

## Historical Evolution

1. **CRT era**: 4:3 dominated (VGA, SVGA, XGA)
2. **~2000**: LCD adoption, shift to 16:10
3. **~2005-2010**: Consumer displays moved to 16:9
4. **2013**: First consumer 4K displays
5. **2013+**: Ultrawide "21:9" formats emerged
6. **2020s**: 16:10 revival in productivity laptops
