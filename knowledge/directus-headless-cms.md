# Directus

Open-source headless CMS and data platform.

## Key Features (as of 11.2)

### Storage & Uploads
- TUS (resumable uploads) support across multiple storage adapters:
  - AWS, Local (since 10.13.0)
  - Supabase, Azure, Cloudinary, GCS (since 11.2)

### AI Integrations
- **AI Web Scraper** - Uses Firecrawl API to scrape web pages and return structured data
- **AI Writer** - Extended to support multiple AI providers and models

### Visualization Charts
- Tree Map Chart - Clustered boxes sized by value, with category grouping
- Funnel Chart - Ascending/descending funnel visualization
- Scatter Plot Chart - 2-axis dot plot with optional axis labels
- Timeline Chart - Gantt-style tasks/events with start/end dates, category grouping

### Content Management
- Content versioning improvements (11.1.2)
- Comments system with API endpoints (11.1.2)
- Customizable Tabular Layout Boilerplate for extensions
