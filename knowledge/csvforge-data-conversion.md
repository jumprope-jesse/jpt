# CSVForge - Browser-Based Data Conversion

https://csvforge.com/

Browser-based CSV conversion suite that processes data locally (no server upload).

## Key Features
- **Multi-format import**: CSV, TSV, XLSX, XLS, TXT with auto format/encoding detection
- **Automatic field detection**: Smart data type detection (text, number, date, boolean)
- **Column operations**: Split, combine, rename, reorder (drag-and-drop)
- **Find & replace**: Supports regex
- **Export formats**: CSV, JSON, XML with proper formatting
- **Live search/sort**: Filter and sort across all data
- **Data comparison**: Highlight differences between datasets

## Privacy
All processing happens in-browser - data never leaves your machine.
