# Django.WTF - Django Package Index

A curated index for discovering Django packages.

## URL
https://django.wtf/

## Features
- **Trending Packages** - See what's gaining traction in the Django ecosystem
- **Top Packages** - Popular, established packages
- **Social Media News** - Aggregated Django discussions from social platforms

## Use Cases
- Discovering new packages for Django projects
- Staying current with Django ecosystem trends
- Finding community discussions and best practices
