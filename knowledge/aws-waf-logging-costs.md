# AWS WAF Free Logging Tier

*Source: https://aws.amazon.com/about-aws/whats-new/2025/09/aws-waf-free-vended-logs-request-volume/ - Added: 2026-01-18*

## Free Tier

AWS WAF now includes **500 MB of CloudWatch Logs Vended Logs Ingestion for every 1 million WAF requests** at no additional cost.

## Key Points

- Allocation applies automatically based on WAF request usage at month end
- Free allocation spans WAF-specific Vended Logs to CloudWatch, S3, and Firehose
- Usage beyond the free tier charged at WAF-specific Vended Logs pricing

## Use Cases

- Security analysis and compliance
- Troubleshooting web application issues
- Monitoring traffic patterns with CloudWatch Log Insights queries
- Anomaly detection dashboards
