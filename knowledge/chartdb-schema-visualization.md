# ChartDB - Database Schema Visualization

## Overview

ChartDB is a web-based tool for visualizing database schemas as diagrams. Useful for understanding complex data structures and communicating database architecture to non-technical stakeholders.

## Links

- App: https://app.chartdb.io
- Example diagram: https://app.chartdb.io/diagrams/y6ikc402oqb6p0wo8uznvc0rt

## Use Cases

- Visualizing existing database schemas
- Documentation for complex data models
- Stakeholder communication
- Database design planning

## Notes

- Web-based, no local installation required
- Can help manage time by quickly grasping database structure instead of reading SQL definitions
