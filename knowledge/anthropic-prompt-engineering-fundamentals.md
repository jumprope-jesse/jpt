# Anthropic Prompt Engineering Interactive Tutorial (PEIT)

Foundational course on prompt engineering with Claude, covering basics to advanced techniques.

*Source: https://thenameless.net/astral-kit/anthropic-peit-00 (adapted from Anthropic's original Google Sheets) - Added: 2026-01-18*

---

## Course Overview

**Target Audience:** Individuals with basic AI understanding who want to learn optimal prompting. Familiarity with technical terms (APIs) required.

**Learning Goals:**
1. Master the basic structure of effective prompts
2. Recognize common failure modes and '80/20' techniques to address them
3. Understand Claude's strengths and weaknesses
4. Build strong prompts from scratch for common use cases

**Model Used:** Claude 3 Haiku (smallest/fastest). Two larger models available: Claude 3 Sonnet and Claude 3 Opus (most intelligent).

---

## Course Structure

9 chapters with exercises + appendix for advanced methods. Intended to be completed in order.

## Key Concepts (from summary)

1. **80/20 Techniques** - Common pitfalls have proven solutions that cover most cases
2. **Strengths/Weaknesses** - Understanding Claude's capabilities enables better prompt design
3. **Use Case Patterns** - Specific domains (legal, coaching) have established prompt patterns
