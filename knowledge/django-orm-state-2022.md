# Django ORM State (2022)

DjangoCon US 2022 keynote by Simon Charette on the state of Object-Relational Mapping in Django.

**Video**: https://www.youtube.com/watch?v=HNlGFrIBl8o

## Key Themes

- Evolution and current state of ORM practices in Django
- Balance between convenience and performance
- Best practices and common pitfalls in ORM usage

## Takeaways

- Understanding ORM internals enables efficient database interactions
- ORM convenience often trades off against performance - know when each matters
- Community feedback drives ORM improvements in Django

## Context

Simon Charette is a Django core contributor who has worked extensively on the ORM, migrations, and database backends. His perspective represents deep knowledge of Django internals.

## Related Django ORM Topics

- QuerySet optimization and lazy evaluation
- N+1 query problems and `select_related`/`prefetch_related`
- Raw SQL vs ORM trade-offs
- Database-specific features and limitations
