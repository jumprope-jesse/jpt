# AWS AppFabric

AWS service that connects SaaS applications for management, security, and productivity.

## Two Facets

1. **AppFabric for Security** - Streamlined management and security of SaaS apps
   - Connects multiple SaaS apps with no coding required
   - Provides standard schema for IT/security teams
   - Normalizes audit logs across applications

2. **AppFabric for Productivity** (preview) - Generative AI for everyday tasks
   - Helps employees complete tasks faster using AI
   - Cross-application context
   - Embeds directly into application UI or AI assistants
   - Provides APIs for generative AI-driven insights based on context from other apps
   - Maintains consistent UX while surfacing relevant context
   - Proactively surfaces actionable insights
   - Generates meeting prep notes, drafts emails, creates tasks, schedules meetings
   - Currently integrates with: Asana, Atlassian Jira, Miro, Slack, Smartsheet, Microsoft 365, Google Workspace
   - Early adopters: Asana, Miro, Smartsheet

## Use Cases

- Centralized SaaS security monitoring
- Normalized audit logging across apps
- Compliance and governance for SaaS tools
- AI-assisted productivity across connected apps

## Notes

- No coding required to connect applications
- Standard schema normalizes data across different SaaS vendors
- Security facet focused on audit log aggregation and normalization
- Productivity facet uses generative AI (still in preview as of 2024)
- Productivity feature announced Nov 2023, available in US East (N. Virginia) region
- Target audience: Application developers looking to enrich app experiences
- Benefits for end users: Access insights without interrupting workflow, increased context for personalized experiences
