# GG Charts - Privacy-Focused Web Diagramming

Open-source, browser-based diagramming tool. No account required, 100% client-side.

**Link:** [gg-charts.com](https://gg-charts.com/)

## Key Features

- Create diagrams with boxes, lists, and connecting lines
- Export to PNG
- Save/load diagrams from disk (as URL-encoded data)
- Light/dark theme toggle
- No ads, analytics, cookies, or data collection

## Icon Libraries

Built-in icons for:
- AWS
- GCP
- Azure
- Kubernetes
- Network diagrams

## When to Use vs Alternatives

| Tool | Best For |
|------|----------|
| **GG Charts** | Quick diagrams, architecture with cloud icons, privacy-sensitive contexts |
| **Mermaid** | Version-controlled diagrams, GitHub/docs integration, text-based |
| **Excalidraw** | Sketch-style diagrams, collaborative whiteboarding |
| **draw.io** | Complex diagrams, extensive shape libraries |

## Notes

- Diagrams stored in URL as encoded data (shareable links contain the full diagram)
- Version 0.9.x as of mid-2024, actively developed
- Good for quick architecture diagrams without signing up for anything
