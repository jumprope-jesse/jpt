# PGlite - Embeddable Postgres

Lightweight, full Postgres database that runs in browser/Node via WASM. Built by Supabase.

## Key Features

- **Lightweight**: Under 3MB gzipped
- **Full Postgres**: Not a subset - actual Postgres compiled to WASM
- **Extensions**: Dynamic loading including pgvector (vectors) and PostGIS (geospatial)
- **Reactive**: Built-in live queries and data sync primitives
- **Local-first**: Designed for offline-first applications

## Use Cases

- Local-first web apps with real SQL
- Testing without Docker/containers
- Browser-based SQL tools and playgrounds
- Embedded analytics in applications
