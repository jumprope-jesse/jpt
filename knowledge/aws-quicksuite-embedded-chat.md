# AWS Quick Suite Embedded Chat

## Overview
Amazon Quick Suite Embedded Chat enables embedding conversational AI directly into applications, combining structured data and unstructured knowledge in a single conversation interface.

**Status**: Generally Available (November 2025)
**Regions**: US East (N. Virginia), US West (Oregon), Asia Pacific (Sydney), Europe (Ireland)
**Pricing**: No additional cost beyond existing Quick Suite pricing

## Key Features

- **Unified Conversational Experience**: Combines structured data (KPIs, analytics) with unstructured knowledge (documents, files) in one conversation
- **Contextual Responses**: Users get answers where they work (CRM, support console, analytics portal) without switching tools
- **Action Capabilities**: Can reference KPIs, pull file details, check feedback, and trigger actions in one conversation

## Integration Options

1. **1-Click Embedding**: Simple integration for quick deployment
2. **API-Based iFrames**: For registered users with existing authentication

## Customization

- Brand colors and styling
- Communication style configuration
- Personalized greetings
- Security controls (scoped data access)

## Connectors

- SharePoint
- Websites
- Slack (send messages)
- Jira (create tasks)

## Use Cases

- CRM systems
- Support consoles
- Analytics portals
- Internal tools requiring contextual AI assistance
