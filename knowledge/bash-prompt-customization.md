# Bash Prompt Customization

Reference for customizing bash prompts with colors, information displays, and creative designs.

*Source: https://www.gilesorr.com/bashprompt/prompts/ - Added: 2026-01-18*

## Overview

Giles Orr's collection of bash prompts, companion to his Linux Documentation Project Bash Prompt HOWTO. Influenced by the defunct BashPrompt Themes Project.

## Prompt Types (Increasing Complexity)

1. **Stock prompts** - RedHat/Suse defaults (user@host, PWD)
2. **Info prompts** - History number, tty, return values, suspended jobs
3. **Colored prompts** - VGA color themes (cyan, blue/yellow, red, gradients)
4. **Load prompts** - One-minute load colorized by severity
5. **Dynamic prompts** - Auto-adjust to terminal width, truncate/fill PWD
6. **Battery prompts** - Laptop power status indicator
7. **Clock prompts** - Upper-right corner clock placement
8. **Title bar prompts** - DOS-style info in pseudo-titlebar
9. **Power prompts** - Feature-rich (slow but inspirational)
