# Aviator - Developer Productivity Platform

**URL:** https://www.aviator.co
**Category:** Developer Tools, CI/CD, Code Review

## Overview

Aviator is a developer productivity platform focused on helping teams "ship faster without breaking things." It provides AI-powered tools for code development, review, and deployment.

## Core Products

### 1. Runbooks
- **Description:** Multiplayer spec-driven development powered by Claude Code
- **Features:** Background agents handle product backlogs, collaborative AI coding
- **Pricing:**
  - Hobby: Free (20k credits/month, max 10 tasks/month)
  - Pro: $10/month (10,000 credits per dollar spent)
  - Enterprise: Custom pricing

### 2. MergeQueue
- **Description:** Scalable merge queue for monorepos
- **Use case:** Keeps builds green at scale by intelligently batching and ordering merges
- **Pricing:**
  - Free: Under 15 users
  - Pro: $12/user/month
  - Enterprise: Custom pricing

### 3. Stacked PRs CLI
- **Description:** Open-source CLI for creating, updating, and reviewing dependent pull requests on GitHub
- **Pricing:** Free and open source
- **Note:** Could be useful for complex feature development with multiple dependent changes

### 4. Releases
- **Description:** Deployment management with PR status tracking
- **Features:** One-click rollbacks, cherry-picks
- **Pricing:**
  - Free: Under 5 users
  - Pro: $8/user/month
  - Enterprise: Custom pricing

### 5. FlexReview
- **Description:** Dynamic code ownership framework
- **Use case:** Reduces review load through automated assignment

## Trial Information

- No payment method required upfront
- Risk-free evaluation period

## Relevance

Potentially useful for:
- Managing PR workflows in larger codebases
- Stacked PRs for complex feature work
- AI-assisted development workflows (Runbooks)
