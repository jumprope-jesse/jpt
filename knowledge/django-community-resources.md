# Django Community Resources

## Django Chat Podcast

**URL**: https://djangochat.com/

A podcast focused on Django and web technology discussions featuring interviews with prominent Django community members.

### Notable Recent Episodes (as of Jan 2024)

**Datasette, LLMs, and Django - Simon Willison**
- Simon Willison is a co-creator of Django
- Currently working on Datasette
- Discusses current state of web technology, AI/LLMs
- Director of the Python Software Foundation
- Episode 154 (Jan 10, 2024)

**Accessibility - Sarah Abderemane**
- Backend Developer at Octopus Energy
- Member of Django Accessibility team
- Work on improving Django docs and admin
- Started Djangonaut Space program
- Organizes Django Paris meetups
- Episode 153 (Dec 20, 2023)

**Understand Django - Matt Layman**
- Senior Staff Software Engineer at Included Health
- Author of "Understand Django" book
- Discusses running Django monolith at scale in healthcare
- Weekly live coding series
- Episode 152 (Dec 6, 2023)

**Kolo for Django - Lily Foote**
- Developer at Klopp Industries
- Working on Kolo, a dynamic code analysis tool for Django
- Recent PRs on Django ORM, serialization, type hints
- Navigator on inaugural Djangonaut Space
- Episode 152 (Dec 6, 2023)

### Key Themes
- Community involvement and development within Django
- Current technologies and tools enhancing Django capabilities
- Accessibility and documentation improvements
- Django at scale in production environments
- AI/LLM integration with Django projects
